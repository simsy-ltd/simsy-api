# SIMSY Public API

> **SIMSY Ltd** · Public · Owner: Alex · [Project Board](https://github.com/orgs/simsy-ltd/projects/1)

Public API specification, SDK, and developer documentation. This repo may be made public for developer onboarding.

---

## What belongs in this repo

- OpenAPI / Swagger specification
- SDK code and client libraries
- API developer documentation
- Integration examples and quickstarts
- API changelog

## What does NOT belong here

- Internal API implementation (use simsy-platform)
- CLI tools (use simsy-cli)
- Any Net4Connect / Net4 information or data

## Related repos

- [simsy-platform](https://github.com/simsy-ltd/simsy-platform)
- [simsy-cli](https://github.com/simsy-ltd/simsy-cli)
- [docs](https://github.com/simsy-ltd/docs)

## Important

**This is a SIMSY-only environment.** Do not add any Net4Connect / Net4 information or data to this repository unless specifically instructed by Alex.
